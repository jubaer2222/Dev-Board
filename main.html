Question-1: What are the different ways to select an element in the DOM?
Use getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll..
Question-2: What is the difference between innerHTML, innerText, and textContent ??
innerHTML gets HTML + text, innerText gets visible text, textContent gets all text.
Question-3: What is event delegation in the DOM?
Add one event listener to a parent to handle events from its child elements.
Question-4: What is event bubbling in the DOM?
An event starts from the target and bubbles up to its ancestors.
Question-5: How do you create, add, and remove elements using JavaScript?
Use createElement, appendChild or append, and remove or removeChild.
